# Calculator App

A simple calculator application built with JavaScript that performs basic arithmetic operations.

## Features
- Perform addition, subtraction, multiplication, and division
- Clear display functionality
- Error handling for invalid expressions
- Real-time display updates

## Technologies
- HTML
- CSS
- JavaScript (DOM manipulation)

## How it works
Users click on calculator buttons to build an expression, which is displayed in the input field. The calculation is performed dynamically using JavaScript when the equals button is pressed.

## Purpose
This project was created as a practice exercise to improve JavaScript fundamentals, including functions, event handling, and basic error handling.
